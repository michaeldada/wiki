JavaScript arithmetic operations are addition, subtraction, multiplication and division. More advanced operations, such as manipulations of percentages, square roots, exponentiation, and logarithmic functions are also, of course, possible in JavaScript. JavaScript arithmetic is performed according to an order of operations.

Below you can see how arithmetic is done in Javascript. *Note that each equation must end with a semi-colon. ";"
